# mojo-utils
Set of basics tools in mojo

## Commands for running mojo container

### Starting the container
```bash
make mjct_start
```

### Removing the container
```bash
make mjct_rm
```

### Restarting the container
```bash
make mjct_restart
```

## Running mojo command

### Mojo execution
